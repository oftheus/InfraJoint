Como usar o analisador

Este módulo sobrepõe uma imagem térmica a uma foto óptica das mãos e mede a temperatura em regiões de interesse (ROIs). Siga os passos abaixo.

1. Carregue os três arquivos

Imagem óptica (foto comum das mãos), matriz térmica (CSV) exportada pela câmera e imagem térmica (o JPEG colorido). Depois clique em Iniciar análise.

2. Alinhe as imagens

O alinhamento automático roda sozinho ao iniciar. Se não ficar bom, use a calibração manual: clique em pelo menos 3 pares de pontos correspondentes na imagem óptica e na térmica e aplique. Ajuste a opacidade para conferir a sobreposição.

3. ROIs manuais

Escolha o formato (círculo ou elipse) e arraste sobre a imagem para desenhar. Clique em uma ROI para selecioná-la; arraste-a para mover e use a alça do canto para redimensionar. Com uma ROI selecionada: setas movem, +/− redimensionam e Delete remove.

4. ROIs articulares (automáticas)

Detectar articulações posiciona 22 ROIs (Punho, MCP e IFP de cada mão) pelos landmarks das mãos. Cada uma pode ser movida e redimensionada como uma ROI manual; Delete (ou o botão restaurar) devolve a ROI à posição detectada. O controle Tamanho das ROIs escala as não ajustadas.

5. Leia as temperaturas

As tabelas mostram mínima, média e máxima em °C. Um ⚠ indica ROI com pouca pele (valor pouco confiável). Cores: ciano = mão esquerda, âmbar = mão direita.

6. Sequência temporal (protocolo)

Na tela inicial, escolha Sequência temporal e importe a pasta da sessão (baseline Est + capturas Din). Cada captura é alinhada e tem as articulações re-detectadas; navegue pela linha do tempo (▶, setas, miniaturas) e abra a Curva de Reaquecimento para acompanhar a temperatura das articulações ao longo do tempo, com o repouso como referência (linha tracejada).

Todo o processamento é feito localmente no navegador, nenhuma imagem ou dado é enviado a servidores.
